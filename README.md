# Sukgo AI — Business Simulation Engine

숙고 (sukgo) — the Korean practice of deep deliberation before an important decision.

10 AI specialists debate your business decision. 1,000 market voices validate demand. Verdict in 60 seconds.

## Live

https://sukgo.ai (coming soon — currently deployed via Vercel; production URL may differ until the domain is cut over.)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
